Swiggy Data Analysis Project

Executive Summary
This project provides a comprehensive analysis of Swiggy's sales and customer rating data for 2025. The goal is to identify growth opportunities, understand customer preferences, and monitor regional performance.

 Key Metrics (KPIs)
- **Total Revenue:** ₹53.01 Million
- **Avg. Customer Rating:** 4.34
- **Total Order Volume:** 197.4K
- **Avg. Price per Item:** ₹268.51

Visualizations Included
1. **Sales Trends:** Monthly, Weekly, and Daily fluctuations.
2. **Product Categories:** Revenue split between Veg and Non-Veg items.
3. **Geographical Performance:** Analysis of Sales by State and City.

How to Run
1. Clone the repo.
2. Install dependencies: `pip install pandas matplotlib seaborn`.
3. Run the analysis: `python analysis.py`.
